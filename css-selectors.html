🎯 Selectores CSS

Selecciona elementos con precisión

Los selectores CSS son patrones que indican qué elementos HTML deben recibir estilos específicos. Dominar los selectores te permite aplicar estilos de manera precisa y eficiente.

Desde selectores básicos hasta combinadores avanzados y pseudo-clases, esta lección cubre todas las formas de seleccionar elementos.

Tipos de Selectores

CSS ofrece múltiples formas de seleccionar elementos:

🎯 Básicos: Elemento, Clase, ID
📋 Atributos: Selección por atributos HTML
🔗 Combinadores: Relaciones entre elementos
✨ Pseudo-clases: Estados especiales (:hover, :nth-child)
🎨 Pseudo-elementos: Partes específicas (::before, ::after)

Cada tipo tiene su propósito y nivel de especificidad.

Selectores Básicos

/* Selector Universal - Todos los elementos */
* {
  margin: 0;
}

/* Selector de Elemento - Por etiqueta HTML */
p {
  color: blue;
}

/* Selector de Clase - Reutilizable */
.mi-clase {
  background: yellow;
}

/* Selector de ID - Único por página */
#mi-id {
  font-weight: bold;
}

Recomendación: Prefiere clases sobre IDs para estilos.

Selectores de Atributo

Seleccionan elementos según sus atributos HTML:

/* Tiene el atributo */
[disabled] { opacity: 0.5; }

/* Atributo con valor exacto */
[type="text"] { border: 1px solid blue; }

/* Atributo que contiene */
[class*="btn"] { padding: 10px; }

/* Atributo que comienza con */
[href^="https"] { color: green; }

/* Atributo que termina con */
[src$=".jpg"] { border: 2px solid red; }

/* Atributo que contiene palabra */
[class~="active"] { font-weight: bold; }
  • [attr] - Tiene el atributo
  • [attr="value"] - Valor exacto
  • [attr*="value"] - Contiene
  • [attr^="value"] - Comienza con
  • [attr$="value"] - Termina con

Combinadores

Seleccionan elementos según su relación con otros:

/* Descendiente (espacio) - Cualquier nivel */
div p { color: blue; }

/* Hijo directo (>) - Solo primer nivel */
ul > li { list-style: none; }

/* Hermano adyacente (+) - Inmediatamente después */
h2 + p { font-weight: bold; }

/* Hermanos generales (~) - Todos después */
h2 ~ p { color: gray; }

Pseudo-clases de Estado

Seleccionan elementos en estados específicos:

/* Estados de interacción */
a:hover { color: red; }
a:active { color: blue; }
a:visited { color: purple; }

/* Estados de formulario */
input:focus { border: 2px solid blue; }
input:disabled { opacity: 0.5; }
input:checked { background: green; }
input:valid { border-color: green; }
input:invalid { border-color: red; }

/* Otros estados */
:target { background: yellow; }
:empty { display: none; }
  • :hover - Mouse sobre el elemento
  • :focus - Elemento enfocado (inputs)
  • :active - Elemento activo (clic)
  • :disabled - Input deshabilitado
  • :checked - Checkbox/radio marcado

Pseudo-clases de Posición

Seleccionan elementos según su posición:

/* Posición específica */
li:first-child { font-weight: bold; }
li:last-child { border-bottom: none; }
li:nth-child(3) { color: red; }

/* Patrones */
li:nth-child(odd) { background: #f0f0f0; } /* Impares */
li:nth-child(even) { background: white; } /* Pares */
li:nth-child(3n) { color: blue; } /* Cada 3 */
li:nth-child(3n+1) { color: green; } /* 1, 4, 7... */

/* Por tipo */
p:first-of-type { margin-top: 0; }
p:last-of-type { margin-bottom: 0; }

/* Único */
p:only-child { text-align: center; }
💡 Diferencia: :nth-child cuenta todos los hijos, :nth-of-type solo del mismo tipo.

Pseudo-elementos

Seleccionan partes específicas de elementos (nota: doble ::):

/* Contenido generado */
.elemento::before {
  content: "→ ";
  color: blue;
}

.elemento::after {
  content: " ✓";
  color: green;
}

/* Primera letra/línea */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

p::first-line {
  font-weight: bold;
}

/* Texto seleccionado */
::selection {
  background: yellow;
  color: black;
}

Pseudo-elementos comunes:

  • ::before - Antes del contenido
  • ::after - Después del contenido
  • ::first-letter - Primera letra
  • ::first-line - Primera línea
  • ::selection - Texto seleccionado

Agrupación de Selectores

Aplica los mismos estilos a múltiples selectores:

/* Agrupación con coma */
h1, h2, h3 {
  font-family: Arial, sans-serif;
  color: #333;
}

/* Múltiples clases (AND) */
.btn.primary {
  background: blue;
  color: white;
}

/* Clase + pseudo-clase */
.link:hover {
  text-decoration: underline;
}

/* Combinación compleja */
nav > ul > li:first-child a:hover {
  color: red;
}
⚠️ Nota: .btn.primary (sin espacio) = elemento con AMBAS clases. .btn .primary (con espacio) = .primary dentro de .btn.

Especificidad de Selectores

Cuando múltiples reglas afectan un elemento, gana la más específica:

Selector Especificidad Puntos
Inline style 1,0,0,0 1000
#id 0,1,0,0 100
.class, [attr], :pseudo-class 0,0,1,0 10
elemento, ::pseudo-element 0,0,0,1 1
p { color: black; } /* 1 punto */
.texto { color: blue; } /* 10 puntos */
#titulo { color: red; } /* 100 puntos */
div p.texto { color: green; } /* 12 puntos */
#titulo.destacado { color: purple; } /* 110 puntos */
💡 Regla: Suma los valores. Mayor especificidad gana. Si empatan, gana el último definido.

Selectores CSS Modernos

Selectores nuevos y útiles:

/* :not() - Negación */
p:not(.especial) { color: gray; }

/* :is() - Simplifica agrupación */
:is(h1, h2, h3) { margin-top: 0; }

/* :where() - Especificidad 0 */
:where(h1, h2, h3) { margin-bottom: 1rem; }

/* :has() - Selector padre (nuevo) */
div:has(> img) { padding: 20px; }

/* :nth-last-child() - Desde el final */
li:nth-last-child(2) { font-weight: bold; }
  • :not() - Excluye elementos
  • :is() - Lista de selectores (especificidad del más alto)
  • :where() - Como :is() pero especificidad 0
  • :has() - Selecciona padre según sus hijos

✅ Pon a Prueba tus Conocimientos

Evalúa tu comprensión de los selectores CSS:

1. ¿Cómo se calcula la especificidad de selectores?

A) Por orden alfabético
B) Inline (1000), IDs (100), clases/atributos/pseudo-clases (10), elementos (1)
C) Todos los selectores tienen la misma especificidad
D) Solo importa cual viene último en el CSS
Respuesta correcta: B) La especificidad se calcula sumando valores: inline styles (1000), IDs (100), clases/atributos/pseudo-clases (10), elementos/pseudo-elementos (1). Mayor especificidad gana.

2. ¿Cuál es la diferencia entre .class1.class2 y .class1 .class2?

A) No hay diferencia
B) .class1.class2 = elemento con AMBAS clases; .class1 .class2 = .class2 dentro de .class1
C) .class1.class2 es un error de sintaxis
D) Solo funciona con divs
Respuesta correcta: B) .class1.class2 (sin espacio) selecciona elementos que tienen AMBAS clases. .class1 .class2 (con espacio) selecciona elementos .class2 que están dentro de elementos .class1.

3. ¿Para qué sirve :nth-child()?

A) Para seleccionar elementos según su posición entre hermanos
B) Solo funciona con listas
C) Selecciona el primer elemento hijo
D) Es lo mismo que :first-child
Respuesta correcta: A) :nth-child() selecciona elementos según su posición. Puede usar números (3), palabras clave (odd, even) o fórmulas (3n+1). Es muy útil para patrones de estilo.

4. ¿Cuál es la diferencia entre pseudo-clase y pseudo-elemento?

A) No hay diferencia
B) Pseudo-clase (:hover) = estado; pseudo-elemento (::before) = parte específica
C) Pseudo-elementos son más modernos
D) Solo los pseudo-elementos usan dos puntos
Respuesta correcta: B) Pseudo-clases (un :) seleccionan elementos en estados especiales (:hover, :focus, :nth-child). Pseudo-elementos (dos ::) seleccionan partes específicas (::before, ::after, ::first-letter).

💪 Ejercicios Prácticos

Refuerza tus conocimientos con estos ejercicios:

Ejercicio 1: Estilizar Listas Anidadas Complejas

Objetivo: Usar selectores avanzados para estilizar listas con diferentes niveles

Requisitos:

  • Crea una lista <ul> con 3 niveles de anidación
  • Estiliza los items del primer nivel (ul > li) con un color
  • Estiliza los items del segundo nivel (ul > li ul > li) con otro color
  • Usa :nth-child(odd) para alternar fondos en items pares/impares
  • Usa :first-child y :last-child para estilos especiales en primer/último item
  • Agrega ::before a items de primer nivel con un icono usando content
  • Usa el combinador + para estilizar el item inmediatamente después de un item con clase .destacado

Desafío Extra: Usa :not() para excluir el último item de cada lista del estilo general

Ejercicio 2: Crear Efectos Hover con Pseudo-clases

Objetivo: Construir una galería con efectos hover usando selectores avanzados

Requisitos:

  • Crea 6 cards en un grid con imágenes
  • Usa :hover para cambiar la escala de la imagen (transform: scale(1.1))
  • Agrega ::after para overlay oscuro que aparece en hover
  • Usa :nth-child(3n) para aplicar color diferente cada 3 cards
  • Estiliza :first-of-type para hacer la primera card más grande
  • Usa el selector de atributo [data-category="featured"] para cards especiales
  • Agrega transiciones suaves (transition: all 0.3s)

Desafío Extra: Usa :has() para cambiar el estilo del contenedor cuando se hace hover en una card

🎉 ¡Domina los Selectores CSS!

Puntos clave para recordar:

  • elemento, .clase, #id - Selectores básicos
  • [attr] - Selección por atributos HTML
  • Combinadores: espacio (descendiente), > (hijo), + (adyacente), ~ (hermanos)
  • Pseudo-clases de estado: :hover, :focus, :active
  • Pseudo-clases de posición: :nth-child(), :first-child, :last-child
  • Pseudo-elementos: ::before, ::after, ::first-letter
  • La especificidad determina qué estilos se aplican
  • Usa :not(), :is(), :has() para selectores avanzados

¡Los selectores precisos hacen el CSS más mantenible!
