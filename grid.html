🎯 Introducción a CSS Grid

Una guía completa para dominar el diseño en cuadrícula

CSS Grid es un sistema de diseño bidimensional que permite crear layouts complejos de manera eficiente y con código limpio.

Con Grid, puedes controlar tanto filas como columnas simultáneamente, creando diseños que antes requerían frameworks complejos o soluciones complicadas.

¿Qué es CSS Grid?

CSS Grid es un sistema de diseño bidimensional que trabaja con filas y columnas al mismo tiempo.

.contenedor {
  display: grid;
}

Conceptos clave:

  • Grid Container: El elemento padre que contiene los grid items
  • Grid Items: Los elementos hijos directos del contenedor
  • Grid Lines: Líneas divisorias que forman la estructura
  • Grid Tracks: Filas o columnas entre dos líneas
  • Grid Cell: Espacio entre 4 líneas (intersección)
  • Grid Area: Área rectangular formada por varias celdas

Activando Grid

Para convertir un elemento en grid container, usa display: grid

.contenedor {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

Grid-Template-Columns

Define el número y tamaño de las columnas:

/* Unidad fr (fraction) - distribuye espacio disponible */
grid-template-columns: 1fr 2fr 1fr;

/* Píxeles fijos */
grid-template-columns: 200px 400px 200px;

/* Repetir */
grid-template-columns: repeat(3, 1fr);

Gap (Espaciado)

Controla el espacio entre filas y columnas:

/* Espacio igual para filas y columnas */
gap: 20px;

/* Diferente para filas y columnas */
gap: 20px 10px; /* row-gap column-gap */

/* Propiedades individuales */
row-gap: 20px;
column-gap: 10px;

Expandir Items

Los items pueden ocupar múltiples columnas o filas:

.item {
  /* Ocupa 2 columnas */
  grid-column: span 2;
  
  /* Ocupa 2 filas */
  grid-row: span 2;
}

Grid-Template-Areas

Define layouts con nombres descriptivos:

.contenedor {
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }

Diseños Responsivos

Grid puede crear layouts responsivos sin media queries:

/* Auto-fit: Ajusta automáticamente el número de columnas */
grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

/* Auto-fill: Llena el espacio con columnas adicionales */
grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));

Casos de Uso Comunes

  • Layouts de páginas: Usa grid-template-areas para estructuras claras
  • Galerías de imágenes: auto-fit con minmax para responsividad
  • Dashboards: Combina grid-column y grid-row para widgets
  • Forms complejos: Alinea labels e inputs en columnas precisas
  • Cards con diferentes tamaños: Usa span para items destacados
  • Layouts de revista: Diseños complejos con areas superpuestas

Grid es ideal para layouts de página completa y diseños bidimensionales complejos.

✅ Pon a Prueba tus Conocimientos

Evalúa tu comprensión de CSS Grid:

1. ¿Qué significa la unidad "fr" en CSS Grid?

A) Fracción de píxeles
B) Fracción del espacio disponible en el grid
C) Frame rate
D) Fixed ratio
Respuesta correcta: B) "fr" significa "fraction" y representa una fracción del espacio disponible en el contenedor grid. 1fr 2fr significa que la segunda columna tendrá el doble de ancho que la primera.

2. ¿Cuál es la diferencia entre grid-template-columns: repeat(3, 1fr) y grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))?

A) No hay diferencia, hacen lo mismo
B) La primera crea exactamente 3 columnas, la segunda ajusta automáticamente según el espacio
C) auto-fit no funciona en Grid
D) repeat(3, 1fr) es más rápido
Respuesta correcta: B) repeat(3, 1fr) crea exactamente 3 columnas fijas, mientras que auto-fit con minmax crea tantas columnas como quepan en el espacio disponible (mínimo 200px cada una), haciendo el diseño responsivo sin media queries.

3. ¿Para qué sirve grid-template-areas?

A) Para crear animaciones en el grid
B) Para nombrar áreas del grid y asignarlas visualmente a elementos
C) Para calcular el área total del grid
D) Para definir el padding de cada celda
Respuesta correcta: B) grid-template-areas permite crear un mapa visual del layout asignando nombres a áreas rectangulares del grid, haciendo el código más legible y semántico (ej: "header header" / "sidebar main").

4. ¿Qué hace grid-column: span 2?

A) Hace que el elemento ocupe 2 filas
B) Hace que el elemento ocupe 2 columnas
C) Crea 2 copias del elemento
D) Agrega 2px de espacio alrededor
Respuesta correcta: B) grid-column: span 2 hace que un elemento grid se extienda a través de 2 columnas. Para expandir filas se usa grid-row: span 2.

💪 Ejercicios Prácticos

Refuerza tus conocimientos con estos ejercicios:

Ejercicio 1: Layout de Blog con Grid Areas

Objetivo: Crear un layout de blog completo usando grid-template-areas

Requisitos:

  • Crea un grid con header (ancho completo), sidebar (1/4), main (3/4) y footer (ancho completo)
  • Usa grid-template-areas para definir la estructura visualmente
  • Agrega gap: 20px entre las secciones
  • Asigna cada sección a su área con grid-area

Desafío Extra: Reorganiza el layout en móvil (sidebar debajo de main) usando media queries

Ejercicio 2: Galería Responsiva Automática

Objetivo: Crear una galería de imágenes que se ajuste automáticamente sin media queries

Requisitos:

  • Usa grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))
  • Crea al menos 8 cards de imágenes
  • Haz que la primera card ocupe 2 columnas (grid-column: span 2)
  • Agrega gap: 15px entre las cards

Desafío Extra: Usa object-fit: cover en las imágenes para que mantengan proporción sin deformarse
