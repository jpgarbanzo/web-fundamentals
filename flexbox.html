🎯 Introducción a Flexbox

Una guía completa para dominar el diseño flexible en CSS

Flexbox (Flexible Box Layout) es un módulo de CSS que permite crear diseños flexibles y responsivos de manera eficiente.

Con Flexbox, puedes organizar, alinear y distribuir elementos dentro de un contenedor, incluso cuando su tamaño es desconocido o dinámico.

¿Qué es Flexbox?

Flexbox es un sistema de diseño unidimensional que trabaja en una sola dirección a la vez: horizontal (fila) o vertical (columna).

.contenedor {
  display: flex;
}

Conceptos clave:

  • Flex Container: El elemento padre que contiene los flex items
  • Flex Items: Los elementos hijos directos del contenedor
  • Main Axis: Eje principal de distribución
  • Cross Axis: Eje perpendicular al principal

Activando Flexbox

Para convertir un elemento en flex container, usa display: flex

.contenedor {
  display: flex;
}

Propiedades de los Flex Items

Flex-grow: Define cuánto puede crecer un elemento

.item { flex-grow: 1; }

Flex-shrink: Define cuánto puede encogerse un elemento

.item { flex-shrink: 1; }

Flex-basis: Define el tamaño base del elemento

.item { flex-basis: 200px; }

Atajo (flex):

.item { flex: 1 1 200px; /* grow shrink basis */ }

Casos de Uso Comunes

  • Centrado perfecto: Usa justify-content: center y align-items: center
  • Barras de navegación: Distribución horizontal con space-between
  • Cards responsivas: Usa flex-wrap: wrap con flex-basis
  • Layouts de aplicaciones: Combina flex-direction: column con flex-grow
  • Galerías de imágenes: Control preciso del espacio y alineación

Flexbox es ideal para componentes individuales y diseños unidimensionales.

✅ Pon a Prueba tus Conocimientos

Evalúa tu comprensión de Flexbox:

1. ¿Cuál es la diferencia principal entre Flexbox y CSS Grid?

A) Flexbox es para layouts 2D, Grid para 1D
B) Flexbox es para layouts 1D (fila o columna), Grid para 2D (filas y columnas simultáneas)
C) Son exactamente iguales
D) Flexbox solo funciona horizontalmente
Respuesta correcta: B) Flexbox trabaja en una dimensión a la vez (fila o columna), mientras que Grid trabaja en dos dimensiones simultáneamente (filas y columnas al mismo tiempo).

2. Si quieres centrar un elemento vertical y horizontalmente en un contenedor, ¿qué propiedades necesitas?

A) display: flex; justify-content: center; align-items: center;
B) display: flex; margin: auto;
C) display: flex; text-align: center;
D) display: block; vertical-align: middle;
Respuesta correcta: A) justify-content controla la alineación en el eje principal y align-items en el eje transversal. Juntos logran el centrado perfecto vertical y horizontal.

3. ¿Qué hace la propiedad flex-wrap: wrap?

A) Cambia la dirección del flex a vertical
B) Permite que los items se ajusten en múltiples líneas cuando no hay espacio
C) Hace que todos los items tengan el mismo tamaño
D) Invierte el orden de los elementos
Respuesta correcta: B) flex-wrap: wrap permite que los flex items se envuelvan en nuevas líneas cuando el espacio del contenedor no es suficiente, creando diseños multi-línea.

4. ¿Cuál es el valor por defecto de flex-direction?

A) column
B) row-reverse
C) row
D) column-reverse
Respuesta correcta: C) Por defecto, flex-direction es row, lo que significa que los elementos se organizan horizontalmente de izquierda a derecha.

💪 Ejercicios Prácticos

Refuerza tus conocimientos con estos ejercicios:

Ejercicio 1: Barra de Navegación Responsiva

Objetivo: Crear una barra de navegación horizontal con logo a la izquierda y menú a la derecha

Requisitos:

  • Usa display: flex en el contenedor principal
  • El logo debe estar a la izquierda y el menú a la derecha (justify-content: space-between)
  • Los items del menú deben estar centrados verticalmente (align-items: center)
  • En móvil, usa flex-direction: column para apilar los elementos

Desafío Extra: Agrega un efecto hover a los links del menú usando transiciones

Ejercicio 2: Galería de Cards Flexibles

Objetivo: Crear una galería de 6 cards que se adapten automáticamente al tamaño de pantalla

Requisitos:

  • Usa flex-wrap: wrap para permitir saltos de línea
  • Cada card debe tener flex-basis: 300px con flex-grow: 1
  • Agrega gap: 20px entre las cards
  • Las cards deben centrar su contenido interno usando Flexbox

Desafío Extra: Usa flex-shrink para evitar que las cards se encojan demasiado en pantallas pequeñas

🎉 ¡Domina Flexbox!

Puntos clave para recordar:

  • Flexbox trabaja en una dimensión: fila o columna
  • Usa justify-content para el eje principal
  • Usa align-items para el eje transversal
  • flex-wrap permite diseños de múltiples líneas
  • Las propiedades de items (flex-grow, flex-shrink) controlan el comportamiento individual
